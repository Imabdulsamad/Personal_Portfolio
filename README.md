# Abdul Samad — Personal Portfolio

A modern, high-performance personal portfolio built to showcase my projects, technical expertise, and developer journey.

## ✨ Features

- ⚡ Built with the latest Next.js App Router
- 🎨 Modern UI with Tailwind CSS and shadcn/ui
- 📱 Fully responsive design
- 🌙 Dark/Light mode support
- 📝 MDX-powered blog
- 🚀 Optimized for performance and SEO
- 📊 Vercel Analytics & Speed Insights

# 🚀 Getting Started

## 1. Clone the Repository

```bash
git clone https://github.com/yourusername/portfolio.git
cd portfolio
```

## 2. Install Dependencies

```bash
npm install
```

## 3. Configure Environment Variables

Create a `.env.local` file in the project root and add the following variables:

```env
# Get a free access key at https://web3forms.com (delivered to the email you register).
NEXT_PUBLIC_WEB3FORMS_ACCESS_KEY=your_web3forms_access_key
NEXT_PUBLIC_SITE_URL=your_domain
```

## 4. Start the Development Server

```bash
npm run dev
```

Open your browser and visit:

```
http://localhost:3000
```

Edit:

```

```

---

### Skills

Edit:

```

### Blog Posts

Create MDX files inside:

```

```

Update blog metadata in:

```

Update your personal information, navigation, social links, and SEO settings in:

# 🚀 Deployment

Deploy effortlessly with **Vercel**.

1. Push your project to GitHub.
2. Import the repository into Vercel.
3. Configure the required environment variables.
4. Deploy.

Your portfolio will automatically include:

- Vercel Analytics
- Speed Insights
- Automatic Deployments
- Global CDN
- SSL
